# ADR 0005: No AWS, Azure, or GCP Collectors in September MVP

## Status

Accepted

## Context

The initial MVP should prove the on-premises VMware, SQL Server, and Oracle discovery path.

## Decision

September MVP excludes AWS, Azure, and GCP collectors.

## Consequences

Positive: clearer scope, faster path to Foundation GA, and better focus on VMware Tax and database modernization.

Tradeoff: cloud-to-cloud assessment waits for a later release.
